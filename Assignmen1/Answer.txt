Ans.In HTML comments use to the create the documentation of the html code. These comments are non-Rendering text of web pages 

example of comment:
    <!DOCTYPE html>
    <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Comments</title>
    </head>
    <body>
    <!--This comment is not Rendering on webpage -->
    </body>
    </html>

Q.5. What is the DOCTYPE Declaration in HTML?
Ans. Doctype HTML is a declaration that tells the browser what 
    version of HTML the document is written in. This declaration appears as the very first line in an HTML file.
